Totally not a static div

I know Python, Javascript, as well as the standard pack of HTML and CSS. Here are a few examples of websites I can create (all websites are hosted by Github Pages):

The Eye Catcher

A little project which is the first one in my series called Simplicity, which includes The Rabbit Hole and The City. It was an attempt to create a 3d cylinder using purely CSS, however, I got a bit carried away and created this shape, after which I added a bit of code and now it changes colours.


The City

The second project in the series Simplicity. A webpage containing a few buildings, where lights randomly turn on and off, as well as a clock which ticks every second. At 8:00 in the morning and evening, the theme of the webpage switches, representing the day night cycle.


The Rabbit Hole

The third and final website in Simplicity cycle. It is a clicker based unfinished story, where you have to click on the circle in the middle to discover new messages, which are stored in a text file and then parsed using Javascript, which means the story can be continued without editing the source HTML file. Some bug fixes are required: the circle does not expand properly on big monitors, but overall, I am happy with this creation.


Endera

This is my unfisihed, wordbuilding project about our world in the future, where we would build moon base, discover other stars and travel far beyond our Solar system. It has a very basic webpage, consisting mainly of text, which has no design whatsoever. It has a few chapters and I was planning a third one, however I got distracted and abandoned it. Maybe one day?
